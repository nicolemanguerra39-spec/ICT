4TH QUARTER OUTPUTS

INTRODUCTION OF JAVASCRIPT

I learned that JavaScript is a programming language used to make websites interactive and dynamic. It helps developers add features such as buttons, animations, and responses that improve the user’s experience on a webpage.

SCRIPTS

I learned that scripts are sets of instructions written in JavaScript that tell the browser what actions to perform. These scripts control how a webpage behaves and allow it to respond to user actions.

CHARACTERISTICS OF JAVASCRIPTS

I learned that JavaScript is a flexible and lightweight programming language that runs directly in a web browser. It is widely used to create interactive features like forms, games, and dynamic content on websites.

EVENTS

I learned that events are actions that occur on a webpage, such as clicking a button, pressing a key, or moving the mouse. JavaScript can detect these events and respond by performing specific functions or tasks.
